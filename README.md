# REVERIE
Turn your favorite songs into sheet music using AI

## HOW IT WORKS
1. **Spleeter** separates vocals and instrumentals.
2. **Basic Pitch** transcribes the instrumental into MIDI + sheet music.

```bash
pip install -r requirements.txt
